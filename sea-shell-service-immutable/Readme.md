# sea-shell-service-immutable

SeaShells service the immutable way using Java records

This service is an illustration on how can we play and workaround publishers and build complex objects.

This emulates real-life situations that may occur in [FIN-TECH](https://www.investopedia.com/terms/f/fintech.asp) organizations like Banks, Finantial Advisors, Insurance Companies and Loans companies.
